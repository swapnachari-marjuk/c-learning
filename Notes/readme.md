# 🚀 C Programming Core Concepts & Tricks (Quick Cheatsheet)

### 📊 C Data Types, Memory Sizes & Format Specifiers

| Data Type                | Memory Size                  | Format Specifier | Range (Approximate)                                         | বিবরণ ও উদাহরণ                           |
| :----------------------- | :--------------------------- | :--------------- | :---------------------------------------------------------- | :--------------------------------------- |
| **`char`**               | **1 Byte** (8 bits)          | `%c`             | `-128` থেকে `127`                                           | সিঙ্গেল ক্যারেক্টার (যেমন: `'A'`, `'Z'`) |
| **`unsigned char`**      | **1 Byte**                   | `%c` / `%u`      | `0` থেকে `255`                                              | ধনাত্মক ক্যারেক্টার / ASCII মান          |
| **`short`**              | **2 Bytes**                  | `%hd`            | `-32,768` থেকে `32,767`                                     | ছোট পূর্ণসংখ্যা                          |
| **`unsigned short`**     | **2 Bytes**                  | `%hu`            | `0` থেকে `65,535`                                           | শুধু ধনাত্মক ছোট পূর্ণসংখ্যা             |
| **`int`**                | **4 Bytes**                  | `%d` / `%i`      | `-2,147,483,648` থেকে `2,147,483,647`                       | সাধারণ পূর্ণসংখ্যা (যেমন: `25`, `-100`)  |
| **`unsigned int`**       | **4 Bytes**                  | `%u`             | `0` থেকে `4,294,967,295`                                    | শুধু ধনাত্মক পূর্ণসংখ্যা                 |
| **`long`**               | **4/8 Bytes** _(OS ভিত্তিক)_ | `%ld`            | $-\approx 2 \times 10^9$ থেকে $\approx 2 \times 10^9$       | বড় পূর্ণসংখ্যা                           |
| **`unsigned long`**      | **4/8 Bytes**                | `%lu`            | `0` থেকে $\approx 4 \times 10^9$                            | শুধু ধনাত্মক বড় পূর্ণসংখ্যা              |
| **`long long`**          | **8 Bytes** (64 bits)        | `%lld`           | $-\approx 9 \times 10^{18}$ থেকে $\approx 9 \times 10^{18}$ | অনেক বড় পূর্ণসংখ্যা                      |
| **`unsigned long long`** | **8 Bytes**                  | `%llu`           | `0` থেকে $\approx 1.8 \times 10^{19}$                       | সবচেয়ে বড় ধনাত্মক পূর্ণসংখ্যা            |
| **`float`**              | **4 Bytes**                  | `%f`             | $1.2 \times 10^{-38}$ থেকে $3.4 \times 10^{38}$             | দশমিক সংখ্যা (যেমন: `3.1416`)            |
| **`double`**             | **8 Bytes**                  | `%lf`            | $2.3 \times 10^{-308}$ থেকে $1.7 \times 10^{308}$           | নিখুঁত দশমিক সংখ্যা                      |
| **`long double`**        | **12/16 Bytes**              | `%Lf`            | $3.4 \times 10^{-4932}$ থেকে $1.1 \times 10^{4932}$         | সায়েন্টিফিক হিসাবের জন্য                 |
| **`char[]`** (String)    | **1 Byte $\times$ Length**   | `%s`             | টেক্সটের দৈর্ঘ্যের ওপর নির্ভর করে                           | ক্যারেক্টার অ্যারে (যেমন: `"Hello"`)     |
| **`void*`** (Pointer)    | **8 Bytes** _(64-bit OS)_    | `%p`             | মেমোরির অ্যাড্রেস                                           | মেমোরির রুম নম্বর (`0x7ffd81`)           |
| **`size_t`** (`sizeof`)  | **8 Bytes** _(64-bit OS)_    | `%zu`            | `0` থেকে বিশাল মান                                          | মেমোরির সাইজ নির্দেশ করতে                |

### ⚠️ মেমোরি সম্পর্কিত ৩টি অতি-জরুরি বিষয়:

1. **`scanf()` ট্র্যাপ (`float` vs `double`):**
   - `printf()`-এ `float` ও `double` উভয়ের জন্যই `%f` কাজ করে।
   - কিন্তু **`scanf()`-এ ইনপুট নেওয়ার সময়** `float`-এর জন্য `%f` এবং `double`-এর জন্য অবশ্যই **`%lf`** দিতে হবে!

2. **`unsigned` এর পাওয়ার:**
   - কোনো টাইপের সামনে `unsigned` বসালে নেগেটিভ মান রাখা বন্ধ হয়ে যায়, ফলে **পজিটিভ মান ধারণের ক্ষমতা দ্বিগুণ হয়ে যায়**।

3. **পয়েন্টার সাইজ:**
   - লিনাক্স ৬৪-বিট সিস্টেমে যেকোনো টাইপের পয়েন্টার (অ্যাড্রেস) মেমোরিতে **৮ বাইট** জায়গা নেয়।

## 2. Constants & Memory Check

- **`const` Keyword:** `const int AGE = 20;` (মানের পরিবর্তন রোধ করে)।
- **Macro (`#define`):** `#define PI 3.1416` (কোডের শুরুতে লিখতে হয়, কোনো `;` দেওয়া যাবে না)।
- **`sizeof()` Operator:** ডাটা টাইপ মেমোরিতে কত Bytes জায়গা নেয় তা মাপতে।

```c
printf("int size: %zu bytes\n", sizeof(int)); // Outputs: 4
```

## 3. Input & Memory Address (`scanf`)

`scanf` ব্যবহারের সময় ভেরিয়েবলের আগে অবশ্যই Address-of Operator (`&`) দিতে হয়। এটি সি-কে ভেরিয়েবলটির আসল RAM Memory Address নির্দেশ করে।

```c
int roll;
scanf("%d", &roll); // '&' is MANDATORY
```

## 4. Input Buffer Clearing Trick ⚠️ (CRITICAL)

#### 🎯 সমস্যা (Problem):

`scanf` দিয়ে ইনপুট নেওয়ার সময় যখন আমরা **Enter** চাপি, তখন মানটি ভেরিয়েবলে চলে যায় কিন্তু **Enter (`\n`)** টি মেমোরির **ইনপুট বাফারে (Input Buffer)** আটকে থাকে।
এর ঠিক পরেই যদি `getchar()` ব্যবহার করা হয়, তবে সে ইউজারের নতুন ইনপুটের জন্য অপেক্ষা না করে বাফারে পড়ে থাকা ওই পুরোনো `\n` টিকেই ইনপুট হিসেবে ধরে নেয়!

#### 🛠️ সমাধান (Solution):

`scanf` এবং `getchar()`-এর ঠিক মাঝে নিচের লাইনটি বসিয়ে বাফারের ময়লা বা `\n` সাফ/ক্লিয়ার করে দিতে হয়:

```c
while (getchar() != '\n'); // Clears leftover '\n' from memory buffer
```

## 5. `getchar()` vs `putchar()`

- **`getchar()`:** কিবোর্ড থেকে একটি মাত্র ক্যারেক্টার গ্রহণ করে।

```c
char grade = getchar();
```

- putchar(): স্ক্রিনে একটি মাত্র ক্যারেক্টার দ্রুত প্রিন্ট করে (printf-এর চেয়ে ফাস্ট)।

```C
putchar(grade);
```

## 6. Operators: C vs JavaScript Differences ⚠️

- **Data Type:** C একটি statically-typed ল্যাঙ্গুয়েজ। C-তে Ternary Operator-এর true এবং false ব্লকের রিটার্ন ভ্যালুর ডেটা টাইপ অবশ্যই একই হতে হবে (অথবা কম্পাইলার যেন সেটিকে নিজে থেকে কনভার্ট করতে পারে)।
- **Integer Division:** `5 / 2` এর ফলাফল সি-তে `2` আসবে, `2.5` নয়! দশমিক পেতে অন্তত একটি সংখ্যাকে ফ্লোটিং পয়েন্ট (`5.0 / 2`) লিখতে হবে।
- **No `===`:** সি-তে Strict Equality (`===`) নেই, শুধু `==` এবং `!=` কাজ করে।
- **Truthy/Falsy Rules:** সি-তে শুধু `0` মানে `False`; যেকোনো অ-শূন্য সংখ্যা (যেমন: `1`, `-10`) মেমোরিতে `True` হিসেবে গণ্য হয়।
- **Ternary Operator (`cond ? exp1 : exp2`):** দুই পাশের টাইপ সামঞ্জস্যপূর্ণ হতে হবে। টাইপ আলাদা হলে ছোট টাইপটি বড় টাইপে অটো-কনভার্ট (Type Promotion) হয়ে যায়।
